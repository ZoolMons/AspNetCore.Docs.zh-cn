如果尚未安装 `dotnet ef`，请安装它作为全局工具：

```dotnetcli
  dotnet tool install --global dotnet-ef
```

有关 EF Core 的 CLI 的详细信息，请参阅 [.Net CLI 的 EF Core 工具引用](/ef/core/miscellaneous/cli/dotnet)。
