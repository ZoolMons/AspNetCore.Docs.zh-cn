* `-c|--content`

  提供内联 HTTP 请求正文。 例如，`-c "{"id":2,"name":"Cherry"}"`。

* `-f|--file`

  提供包含 HTTP 请求正文的文件的路径。 例如，`-f "C:\request.json"`。

* `--no-body`

  指示无需 HTTP 请求正文。
